#### 🐼集约化运维🐼
以六边形盾牌图案为基础的logo有着明显的安全运维平台属性，着重体现surpass的安全属性。六边形如石，纹理绘盾其中，象征着surpass以坚为信，贞刚有质，赋予了它强烈的安全、坚定气质。盾石并列交叠，可谓落落出岫云，浑浑倚天石。这也寓意了surpass是不同凡响，超越进取的全方位运维管理平台。
